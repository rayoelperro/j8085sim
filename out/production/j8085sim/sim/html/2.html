How To Execute A Program?

After the program is entered press the Execute button (if the whole program is to be executed) or Single Line Execution button (if the program is to be executed line by line) on the toolbar. The program is executed only if there are no compilation errors. Errors (if any) are displayed in the Messages box.
The execution stops if any runtime error occurs (like invalid opcode)
To stop the execution press the Stop button. The output of the exection will be reflected on the registers, stack, port and address locations.
The clock speed can be adjusted to adjust the speed of execution.
